> SSH = Secure Shell

SSH keeps disconnecting after a while. You can fix that by sending a `heartbeat` every once in a while. Add this to
`~/.ssh/config`
```
Host *
  ServerAliveInterval 240
```
and also make this file read-able to all (`chmod 600 ~/.ssh/config`)
